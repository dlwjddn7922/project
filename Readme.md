# 문제 정의

- 앞으로의 가장 큰 걱정거리인 취업에 관한 기업리뷰 서비스를 만들어 보고 싶었다.
- 본인이 희망하는 기업을 분석하여 해당 기업의 평가를 쉽게 확인할 수 있다.
- 회사의 장단점과 평점을 찾아 분류하는 프로젝트이다.
